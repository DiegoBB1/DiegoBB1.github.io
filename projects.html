Galaxy's Revival

In my final semester at UTSA, I developed Galaxy's Revival, a 2D space twin-stick shooter game using Unity and C#. In this game, players take on the role of the last remaining soldier of the Galactic Defense Corps, tasked with aiding planets and their inhabitants after a large-scale interplanetary invasion. Players navigate from system to system, battling enemies, rescuing allies, and securing resources to help rebuild the galaxy.

Galaxy's Revival features multiple levels and environments, each with unique objectives, enemy types, and challenging gameplay scenarios. Players can upgrade their abilities and gear, unlocking new weapons, power-ups, and defenses that adapt to their playstyle. The game's variety of objectives and upgrades allows for a wide range of gameplay experiences.

To enhance performance and efficiency, I implemented design patterns like object pooling for managing enemy instances and the state pattern to control complex enemy behaviors. These optimizations ensure smooth gameplay and responsive controls, even with numerous elements on screen.

Cache Simulator

While at UTSA, I developed a Python program to simulate the process of memory caching, designed both as a learning tool and an analytical resource. This cache simulator models various caching mechanisms and provides insights into how different cache configurations impact performance. The program accepts a trace file along with a cache configuration (such as cache size, associativity, and block size) and outputs a detailed statistical analysis, including metrics like hit rate, miss rate, and cost.

This tool was built to enhance my understanding of memory caching concepts and to help users identify optimal cache parameters for specific workloads. By experimenting with different configurations, users can leverage the program to fine-tune cache performance based on their unique requirements, making it a valuable tool for both educational and practical applications.
